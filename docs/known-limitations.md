# Known Limitations

- Repository detection is heuristic and intentionally conservative.
- `package.json` scripts are detected, but arbitrary build systems are not inferred.
- Issue triage is deterministic keyword analysis, not semantic AI analysis.
- Suggested files in plans are estimates and must be verified before editing.
- GitHub Issues are not fetched in the current `0.1.x` releases.
- MaintainerKit does not apply labels, comments, branches, commits, or pull requests.
- Generated files are starting points and require maintainer review.
- Protected-area detection reduces risk but cannot identify every security-sensitive change.
