# Roadmap

## v0.1.x: Foundation

- Validate repository detection across representative TypeScript, Python, Rust, and Go projects
- Improve generated guidance and safety documentation
- Establish reproducible releases and security checks
- Collect honest adoption and case-study evidence

## v0.2.0: Read-only GitHub Integration

- `maintainerkit triage --issue <number>`
- GitHub Issue retrieval with explicit repository selection
- Label and maintainer-comment suggestions without applying changes
- Machine-readable JSON output for integrations

## Later

- Opt-in label and comment application
- Codex work loops with branch and pull request automation
- Agent Readiness Score
- Changelog and release preparation

Write operations will remain disabled by default and require explicit approval.
