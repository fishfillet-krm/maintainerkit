# Support

## Questions And Usage Help

Use [GitHub Discussions](https://github.com/fishfillet-krm/maintainerkit/discussions) for usage
questions, repository-readiness feedback, and examples of MaintainerKit in practice.

## Bugs And Features

Use the repository's issue forms for reproducible bugs and focused feature requests. Include the
MaintainerKit version, operating system, Node.js version, command, and a minimal repository shape
when possible.

## Security

Do not disclose suspected vulnerabilities in a public issue or discussion. Follow
[SECURITY.md](SECURITY.md).

## Support Expectations

MaintainerKit is maintained on a best-effort basis. There is no guaranteed response time or
commercial support commitment.
